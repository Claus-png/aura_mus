## О приложении

**AURA** — это приватный музыкальный плеер для Android, созданный для тех, кто хочет слушать музыку без подписок и ограничений. Приложение позволяет скачивать треки из Яндекс Музыки, Spotify, SoundCloud и YouTube Music напрямую на устройство, управлять локальной библиотекой, создавать плейлисты и гибко настраивать звук через встроенный эквалайзер.

AURA написана на Kotlin с использованием Jetpack Compose и ExoPlayer. Интерфейс полностью на русском языке.

## Возможности

### 🎵 Плеер
- Полноэкранный плеер с обложкой альбома, анимацией и управлением жестами
- Свайп влево/вправо для переключения треков
- Режимы повтора: выкл / все треки / один трек
- Умное перемешивание с учётом истории прослушиваний
- Таймер сна — автоматическая остановка воспроизведения через заданное время
- Мини-плеер всегда доступен в нижней части экрана
- Воспроизведение продолжается в фоне (foreground-сервис)
- Автосохранение позиции — при перезапуске приложения воспроизведение продолжается с того же места

### 📥 Загрузка музыки
Вставьте ссылку на трек или плейлист — приложение автоматически определит источник и скачает музыку.

| Источник | Треки | Плейлисты |
|---|---|---|
| **Яндекс Музыка** | ✅ | ✅ |
| **Spotify** | ✅ (через сервер) | ✅ (через сервер) |
| **SoundCloud** | ✅ (через сервер) | ✅ (через сервер) |
| **YouTube Music** | ✅ (через сервер) | ✅ (через сервер) |

- Прогресс загрузки отображается в реальном времени
- Статусы: ожидание → загрузка → теггинг → готово / ошибка / отменено
- Автоматическая нормализация громкости (ReplayGain через ffmpeg)
- Выбор качества: от 32 до 320 кбит/с или авто (максимум)
- Треки сохраняются во внешнее хранилище (`Android/data/.../Music/<источник>/`)

### 📚 Библиотека
- Автоматическое сканирование и индексирование загруженных треков
- Метаданные: название, исполнитель, альбом, год, жанр, номер трека, длительность
- Обложки альбомов извлекаются и кэшируются локально
- Поддержка тегов ID3 / Vorbis Comment
- Импорт папки с локальными аудиофайлами прямо с устройства
- SHA-256 дедупликация — один и тот же трек не добавится дважды
- Статистика библиотеки: количество треков, общий размер (МБ), общая продолжительность (ч)

### 📋 Плейлисты
**Пользовательские плейлисты** — создавайте, переименовывайте, удаляйте, добавляйте и переупорядочивайте треки.

**Умные (динамические) плейлисты** — формируются автоматически на основе статистики:

| Плейлист | Описание |
|---|---|
| **Библиотека** (название можно изменить) | Все треки |
| **Часто слушаю** (название можно изменить) | Треки с высоким числом прослушиваний |
| **Редкие треки** (название можно изменить) | Треки, которые вы почти не слушаете |
| **Избранное** | Треки, добавленные в избранное |
| **С рейтингом** | Треки, которым вы поставили оценку |
| **Недавно добавленные** | Последние загрузки |
| **Недавно слушал** | По дате последнего воспроизведения |

Названия умных плейлистов можно переименовать прямо в настройках.

### 🎛 Эквалайзер
- 5-полосный параметрический эквалайзер (60 Гц, 230 Гц, 910 Гц, 3 кГц, 14 кГц)
- Диапазон: от −15 дБ до +15 дБ на каждой полосе
- Bass Boost (усиление баса): 0–1000
- Virtualizer (объёмный звук): 0–1000
- 10 встроенных пресетов:

| Пресет | Описание |
|---|---|
| Flat | Без изменений |
| Bass Boost | Усиленные низкие частоты |
| Треки | Мягкое усиление для pop-музыки |
| Rock | Насыщенные низкие и высокие |
| Pop | Подчёркнутые средние частоты |
| Jazz | Тёплый, детальный звук |
| Hip-Hop | Акцент на басах |
| Electronic | Динамичный, широкий звук |
| Classical | Объёмный, воздушный |
| Vocal | Подчёркнутый вокал |

- Настройки эквалайзера сохраняются между перезапусками
- EQ применяется в реальном времени (Android AudioEffect API)

### 🎧 Bluetooth-профили
- Приложение запоминает, какие настройки EQ вы использовали с каждым Bluetooth-устройством
- При повторном подключении наушников/колонки — настройки EQ применяются автоматически
- Можно отключить сбор профилей в настройках

### 🖥 Домашний сервер
Для загрузки из Spotify, SoundCloud и YouTube Music требуется собственный сервер с установленными `yt-dlp` / `spotdl`. Приложение подключается к нему по URL.

- Статус сервера (онлайн/оффлайн) всегда виден в настройках
- Настройка и проверка соединения в один клик

### 🎤 Тексты песен
- Поддержка встроенных текстов (plain text)
- Поддержка синхронизированных текстов в формате LRC (слова подсвечиваются по времени)
- Текст отображается прямо в экране плеера — смахни вверх, чтобы открыть

### 📊 Статистика прослушиваний
- Счётчик воспроизведений для каждого трека
- Дата последнего прослушивания
- Рейтинг треков (пользовательская оценка)
- На основе статистики работают умные плейлисты и алгоритм перемешивания
- Статистику можно сбросить (с двойным подтверждением)
- Сбор статистики можно полностью отключить

## Установка

1. Скачайте APK-файл из официального источника (канал автора)
2. На устройстве разрешите установку из неизвестных источников:  
   **Настройки → Безопасность → Установка неизвестных приложений**
3. Установите APK
4. При первом запуске выдайте разрешения на доступ к хранилищу (необходимо для сохранения музыки)

**Минимальная версия Android:** 10 (API 29)  
**Рекомендуемая:** Android 12+

## Первый запуск

### Шаг 1 — Настройка прокси (если нужен)
Если вы находитесь в регионе, где доступ к Яндекс Музыке или другим сервисам ограничен:

1. Откройте **Настройки** (иконка шестерёнки на главном экране)
2. Перейдите в раздел **Прокси**
3. Выберите тип: SOCKS5, HTTP, VLESS или MTProto
4. Введите адрес сервера и порт (или отсканируйте QR-код)
5. Включите прокси переключателем

### Шаг 2 — Вход в Яндекс Музыку
Для загрузки треков с Яндекс Музыки требуется авторизация:

1. В **Настройках** нажмите **«Войти в Яндекс Музыку»**
2. Войдите в свой аккаунт через браузер
3. После успешного входа логин отобразится в настройках

> Без авторизации Яндекс Музыка отдаёт только 30-секундные превью вместо полных треков.

### Шаг 3 — Настройка домашнего сервера (для Spotify / SoundCloud / YouTube)
1. Разверните сервер с `yt-dlp` / `spotdl` на любом VPS или домашнем компьютере
2. В **Настройках** нажмите **«Настроить»** в разделе **Домашний сервер**
3. Введите URL вашего сервера
4. Дождитесь статуса **«Сервер подключён»**

### Шаг 4 — Добавьте музыку
**Вариант А — Скачать из интернета:**
1. Перейдите на вкладку **«Добавить музыку»**
2. Вставьте ссылку на трек или плейлист
3. Нажмите кнопку загрузки
4. Следите за прогрессом в списке заданий

**Вариант Б — Импорт локальных файлов:**
1. На главном экране нажмите **«Импортировать папку»**
2. Выберите папку с аудиофайлами на устройстве
3. Треки появятся в библиотеке автоматически

## Часто задаваемые вопросы

**Почему Яндекс Музыка скачивает только 30 секунд?**  
Это происходит, если вы не авторизованы или сессия истекла. Войдите заново через **Настройки → Войти в Яндекс Музыку**.

**Почему плейлисты «Часто слушаю» и «Редкие треки» пустые?**  
Эти плейлисты формируются на основе статистики прослушиваний. Убедитесь, что в настройках включена опция **«Отслеживать прослушивания»**.

**Почему не загружается Spotify / SoundCloud?**  
Для этих источников нужен настроенный домашний сервер. Проверьте статус в **Настройки → Домашний сервер**.

**Треки не появляются в библиотеке после загрузки.**  
Скорее всего, загрузка ещё не завершена. Откройте вкладку **«Добавить музыку»** и проверьте статус задания.

**Как изменить название умного плейлиста?**  
Долгое нажатие на плейлист → **Переименовать**.
